# Documentation Update Rule

**CRITICAL DIRECTIVE FOR ALL FUTURE AI AGENTS:**
At the conclusion of your assigned tasks, or at the end of a significant development iteration, you MUST update the following markdown files in the project root to reflect the current state of the project:

1. `AGENT_HANDOFF.md`: Add a summary of the features, architectural shifts, or endpoint changes you made so the next agent understands the project's exact state.
2. `LESSONS_LEARNED.md`: If you encounter any bugs, crashes, or logical errors during your iteration, document them here in simple English along with how you fixed them.

This ensures institutional memory is preserved across different conversation contexts.
